क्या है?

Computer एक electronic machine है जो:

Input लेता है
Data को process करता है
Output देता है
Information को store भी करता है
🔹 Computer के मुख्य भाग (Main Components)
1. Input Devices (इनपुट डिवाइस)

👉 जिनसे हम data देते हैं

Keyboard
Mouse
Scanner
Microphone
2. CPU (Central Processing Unit)

👉 इसे Computer का brain कहा जाता है

CPU के 3 भाग होते हैं:

ALU (Arithmetic Logic Unit) → Calculation करता है
CU (Control Unit) → सभी काम control करता है
Registers → Temporary data store करते हैं
3. Memory (मेमोरी)
Primary Memory
RAM (Temporary memory)
ROM (Permanent memory)
Secondary Memory
Hard Disk
SSD
Pen Drive
4. Output Devices (आउटपुट डिवाइस)

👉 जिनसे result मिलता है

Monitor
Printer
Speaker
🔹 Computer कैसे काम करता है? (IPO Cycle)
Input → Data देना
Process → CPU काम करता है
Output → Result मिलता है
🔹 Computer के प्रकार (Types of Computer)
1. Based on Size
Microcomputer (PC, Laptop)
Minicomputer
Mainframe
Supercomputer
2. Based on Work
Analog Computer
Digital Computer
Hybrid Computer
🔹 Computer की विशेषताएँ (Features)
⚡ Fast (बहुत तेज)
🎯 Accurate (सटीक)
💾 Storage (डेटा स्टोर करता है)
🔁 Automation (अपने आप काम करता है)
🤖 No feelings (कोई भावना नहीं)
🔹 Computer के उपयोग (Uses)
Education
Banking
Office work
Internet
Entertainment
